## Design Context

### Users
- **Primary**: Developers and platform engineering teams managing API keys across services
- **Free tier**: Solo devs and founders securing side projects and startups
- **Pro/Enterprise**: Engineering teams managing secrets across multiple environments and services
- **Context**: Using VaultProxy within their deployment workflow — they want to set it up, forget about it, and trust it's working. Speed and clarity are paramount.
- **Job to be done**: Mask, rotate, and sync API keys without redeploying. The dashboard exists to manage that — nothing more.

### Brand Personality
- **3 words**: Invisible, trustworthy, precise
- **Voice**: Terse, confident, no filler. Says what it needs to say and stops.
- **Emotional goal**: Calm confidence. Users should feel like their secrets are in good hands without the UI drawing attention to itself. The product works; the interface stays out of the way.
- **Anti-references**: Overly branded SaaS dashboards, dark "hacker" aesthetics, heavy illustration, gamified interfaces, identical card grids, centered empty states, decorative icons

### Aesthetic Direction
- **Visual tone**: Minimal, warm, premium — the Stripe school of invisible design
- **References**:
  - **Stripe dashboard** — Typography-driven hierarchy, generous white space, the UI disappears behind the content
  - **getlate.dev (Zernio)** — Warm accent color, cream backgrounds, breathable layout, modern but not cold
- **Theme**: Light mode only. Warm neutrals (cream/off-white), never cool grays.

### Design System (Implemented)

#### Color Palette
Defined in `globals.css` via Tailwind v4 `@theme`. All colors are warm-tinted — no cool grays, no default Tailwind semantic colors.

| Scale | Role | Range |
|-------|------|-------|
| `cream-50` to `cream-900` | Neutral scale | `#FDFCF9` (bg) to `#1A1815` (text) |
| `terra`, `terra-hover`, `terra-subtle` | Primary accent (terracotta) | `#C74B28`, `#A93F21`, `#F9EBE6` |
| `sage-50` to `sage-800` | Success states | `#F0F4EC` to `#374F2F` (mossy green) |
| `ember-50` to `ember-700` | Error/danger states | `#FAE8E5` to `#972E28` (warm red) |

**Rules**: Never use default Tailwind `green-*`, `red-*`, `gray-*`, or `indigo-*`. Always use the custom warm palette.

#### Typography
- **Sans**: Plus Jakarta Sans (400, 500, 600, 700) via `next/font/google`, variable `--font-jakarta`
- **Mono**: JetBrains Mono (400, 500) via `next/font/google`, variable `--font-jetbrains`
- **Hierarchy**: h1 `text-2xl font-bold`, h2 `text-base font-semibold`, h3 `text-sm font-semibold`, body `text-sm`, caption `text-xs`
- **Brand mark**: `text-[11px] font-bold text-terra tracking-[0.2em] uppercase`

**Rules**: Never use Inter, Roboto, system defaults, or overused fonts. No monospace-as-aesthetic for non-code text.

#### Layout Patterns
- **Sidebar**: Text-only nav (no icons), 224px (`w-56`), `border-r border-cream-200`
- **Active nav**: `text-terra font-medium` only — no background fill
- **Content area**: `px-4 py-6 md:px-10 md:py-8`, constrained to `max-w-4xl`
- **Mobile**: Hamburger header + slide-in sidebar overlay (`animate-slide-in`)
- **Stats**: Inline text with a prominent number, not card grids

#### Interaction Patterns
- **Empty states**: Show the form directly (the empty state IS the action). No decorative icons, no cards wrapping "nothing here" text. Left-aligned, never centered.
- **Form reveals**: `animate-fade-in` (200ms ease-out)
- **Delete confirmation**: Inline "Delete? Yes / No" — no `window.confirm()`, no modals
- **Buttons**: Primary `bg-terra text-white`, secondary/cancel is text-only `text-cream-600 hover:text-cream-800`
- **Inputs**: `bg-white border border-cream-300 rounded-md`, focus: `ring-2 ring-terra/20 border-terra`

#### Motion
- `fade-in`: 200ms ease-out, opacity + translateY(4px)
- `slide-in`: 200ms ease-out, translateX(-100% to 0) for mobile sidebar
- All animation respects `prefers-reduced-motion: reduce`

### Design Principles

1. **Invisible by default** — The best interface is one you don't notice. Reduce chrome, increase content density, let data speak. Text-only nav. Inline stats. No decorative icons.
2. **One job, done well** — Every screen serves a single purpose. Empty states become the action (show the form). "Get started" disappears once setup is complete.
3. **Warm precision** — Cream neutrals + terracotta accent. Sage for success, ember for errors. The entire color system is temperature-consistent — no cold Tailwind defaults leak through.
4. **Trust through restraint** — Security products earn trust by being calm and predictable. No shields, no locks, no "bank-grade" copy. Inline delete confirmations instead of modal dialogs.
5. **Typography over decoration** — Plus Jakarta Sans (bold headings, semibold sections, regular body) and JetBrains Mono (code, aliases, keys) carry the hierarchy. When in doubt, remove visual noise rather than adding decoration.
